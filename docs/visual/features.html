sequenceDiagram participant STU as Student participant BE as Backend participant DB as MySQL participant EMAIL as Email Service participant SUP as Supervisor STU->>BE: Submit request BE->>BE: Generate unique token BE->>DB: Save token + expiry BE->>EMAIL: Send approval link EMAIL->>SUP: Email with link SUP->>BE: Click link (validate token) BE->>DB: Mark approved BE-->>SUP: Show success page

flowchart TD ENV["ENABLE_AGENTS=true"] --> AM["AgentManager"] AM --> A1["eligibilityUpdater\n(periodic)"] AM --> A2["projectDeadlineMonitor\n(hourly)"] AM --> A3["internshipLifecycleMonitor\n(hourly)"] AM --> A4["deadlineReminderAgent\n(daily 02:00)"] AM --> A5["tokenCleanupScheduler\n(daily)"] A1 --> DB[(MySQL)] A2 --> DB A3 --> DB A4 --> NS["NotificationService"] A5 --> DB NS --> DB

Agent Details

Agent Type Schedule Description
eligibilityUpdater Scheduler Periodic Eligibility Checker — ตรวจสอบสิทธิ์นักศึกษาตาม unlock conditions
projectDeadlineMonitor Scheduler Hourly Project Purge — ตรวจสอบ deadline โครงงาน/ปริญญานิพนธ์
internshipLifecycleMonitor Monitor Hourly Document Status — ติดตาม lifecycle ฝึกงาน
deadlineReminderAgent Scheduler Daily 02:00 Deadline Reminder — ส่ง notification ก่อน deadline
tokenCleanupScheduler Scheduler Daily ลบ expired tokens + blacklisted tokens
academicSemesterUpdater Scheduler Periodic Academic Semester Auto-update — อัปเดตปีการศึกษา/เทอมอัตโนมัติ
logbookQualityMonitor Monitor Periodic Logbook Quality — ตรวจสอบคุณภาพบันทึก logbook
securityMonitor Monitor Periodic Security Monitor — ตรวจจับ anomalies และ suspicious activity

ENABLE_AGENTS=true
ACADEMIC_AUTO_UPDATE_ENABLED=true

PDF System

ระบบ generate เอกสารราชการ 3 ประเภท ด้วย pdfkit + TH Sarabun New + ตราครุฑ

Letter

หนังสือขอความอนุเคราะห์

  • To employer
  • Header: department address + doc number (อว.)
  • Subject line bold official format
  • Footer: contact ext. 4601
  • ตราครุฑ positioned absolutely
Transmission

หนังสือส่งตัว

  • Formal student transmission
  • 3-item checklist + notes
  • Department head signature (departmentInfo.js)
  • Faculty & university below
  • Correct Content-Disposition header
Certificate

ใบรับรองการฝึกงาน

  • After 250+ hours
  • Total score / FULL_SCORE (100)
  • PASS_SCORE displayed
  • Loma font for Thai
  • IDOR-protected

Technology: pdfkit + TH Sarabun New font + ตราครุฑ seal
