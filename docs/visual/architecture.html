Nginx Routing

Reverse proxy configuration mapping URLs to internal services

Internet → Nginx (port 80)
  ├── /           → Frontend (127.0.0.1:3000)
  ├── /api/       → Backend  (127.0.0.1:5000)
  ├── /socket.io/ → Socket.io (127.0.0.1:5000)
  ├── /api-docs   → Swagger  (127.0.0.1:5000)
  └── /uploads/   → Static files
