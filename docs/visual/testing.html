Solution 01

Thai Font ใน PDF

Added TH Sarabun New + Loma font in Docker image & configured pdfkit to use them

Problem 02

IDOR Vulnerability

Solution 02

IDOR Vulnerability

Added ownership check on 6 PDF endpoints + automated security tests to prevent regression

Problem 03

Workflow State Complexity

Solution 03

Workflow State Complexity

Solution 04

MySQL UTF-8mb4 on Docker
