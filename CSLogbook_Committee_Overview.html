CSLogbook

ระบบจัดการกิจกรรมนักศึกษา — ภาควิชาวิทยาการคอมพิวเตอร์และเทคโนโลยีสารสนเทศ, มจพ.

CSLogbook คือระบบ Web Application ที่รวมการจัดการฝึกงานและโปรเจกต์จบการศึกษาไว้ในที่เดียว ครอบคลุมตั้งแต่การลงทะเบียน, ส่งเอกสาร, เขียน Logbook, นัดพบอาจารย์, ไปจนถึงการสอบและประเมินผล — แทนที่กระบวนการกระดาษและ Excel ที่กระจัดกระจาย

🔍 ปัญหาที่ระบบแก้ไข

❌ ก่อนใช้ CSLogbook

  • 📄 เอกสารกระจาย — ส่งผ่าน Email, Line, กระดาษ
  • 📊 ติดตามความคืบหน้ายาก — อาจารย์ต้องถามทีละคน
  • ⏰ Deadline ตกหล่น — ไม่มีระบบแจ้งเตือนอัตโนมัติ
  • 🔄 ข้อมูลซ้ำซ้อน — กรอกซ้ำหลายที่
  • 📈 ไม่มี Dashboard สรุปภาพรวม

✅ หลังใช้ CSLogbook

  • 📱 ศูนย์กลางเดียว — ทุกเอกสารและขั้นตอนอยู่ในระบบ
  • 📊 Dashboard แบบ Real-time — ดูสถานะนักศึกษาทั้งหมดได้ทันที
  • 🔔 แจ้งเตือนอัตโนมัติ — ก่อน Deadline, เมื่อมีเอกสารใหม่
  • ⚡ กรอกครั้งเดียว — ข้อมูลนักศึกษาใช้ร่วมทั้งระบบ
  • 📈 รายงานสรุปอัตโนมัติ — ดาวน์โหลดได้ทันที

⭐ ความสามารถหลักของระบบ

📋

จัดการฝึกงาน

ลงทะเบียน, หนังสือขอความอนุเคราะห์ (CP05), หนังสือส่งตัว, ส่ง Logbook รายสัปดาห์, ขอใบรับรอง, ประเมินผล

🎓

จัดการโปรเจกต์จบ

สอบหัวข้อ, System Test, KP02, Defense, ติดตาม Milestone, บันทึกการประชุม, ส่งเอกสารเล่มจบ

📄

Workflow เอกสาร

นักศึกษาส่ง → อาจารย์ตรวจ → อนุมัติ/ส่งกลับแก้ไข พร้อมลิงก์อนุมัติทาง Email ได้ทันที

🔔

แจ้งเตือนอัตโนมัติ

แจ้งเตือนแบบ Real-time เมื่อมีเอกสารใหม่, ใกล้ Deadline, หรือมีการเปลี่ยนสถานะ

📊

Dashboard & Reports

ภาพรวมสถานะนักศึกษาทั้งหมด, รายงานความคืบหน้า, สถิติฝึกงาน, workload อาจารย์

📅

ปฏิทิน & Deadline

ปฏิทินรวม Deadline ทั้งหมด, กำหนดส่งแบบยืดหยุ่น, ล็อกเอกสารหลังกำหนด

🔄 กระบวนการทำงานหลัก
