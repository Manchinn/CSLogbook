# Flowchart ระบบฝึกงานบนเว็บไซต์ (อัปเดตล่าสุด พร้อมระบุผู้ดำเนินการ)

```mermaid
flowchart TD
    A[เริ่มต้น (นักศึกษา)] --> B[ค้นหาและเลือกบริษัท (นักศึกษา)]
    B --> C[กรอก คพ.05 ออนไลน์ (นักศึกษา)]
    C --> D[ระบบส่งให้หัวหน้าภาคพิจารณา (ระบบ)]
    D --> E{อนุมัติโดยหัวหน้าภาค?}
    E -->|ไม่อนุมัติ| C
    E -->|อนุมัติ| F_StaffAction[เจ้าหน้าที่ออกหนังสือขออนุเคราะห์ฝึกงาน (เจ้าหน้าที่ภาควิชา)]
    F_StaffAction --> F[ดาวน์โหลดเอกสารขอความอนุเคราะห์ (นักศึกษา)]
    F --> G[รอการตอบรับจากบริษัท (นักศึกษา/บริษัท)]
    G --> H[อัปโหลดหนังสือตอบรับ (นักศึกษา)]
    H --> I[รับเอกสารฝึกงาน(หนังสือส่งตัว)ที่ภาควิชา (นักศึกษา จาก เจ้าหน้าที่ภาควิชา)]
    I --> J[เริ่มฝึกงาน (นักศึกษา)]
    J --> K[บันทึกการฝึกงานประจำวัน (นักศึกษา)]
    K --> L[สิ้นสุดการฝึกงาน (นักศึกษา)]
    L --> M[ส่งสรุปผลการฝึกงาน (นักศึกษา)]
```
