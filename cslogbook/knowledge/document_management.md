# ระบบจัดการเอกสาร CSLog

## 1. เอกสารโครงงานพิเศษ (Project Documents)

### แบบฟอร์มออนไลน์
1. **คพ.01 - แบบฟอร์มเสนอหัวข้อโครงงานพิเศษ**
   - ข้อมูลนักศึกษา
     * รหัสนักศึกษา
     * ชื่อ-นามสกุล (ไทย/อังกฤษ)
     * สาขาวิชา
     * เบอร์โทรศัพท์
     * อีเมล
   - ข้อมูลโครงงาน
     * ชื่อโครงงาน (ไทย/อังกฤษ)
     * ประเภทโครงงาน (Software/Hardware/Research)
     * วัตถุประสงค์
     * ขอบเขตการดำเนินงาน
     * เครื่องมือและอุปกรณ์ที่ใช้
     * ประโยชน์ที่คาดว่าจะได้รับ
   - อาจารย์ที่ปรึกษา
     * ชื่อ-นามสกุล
     * ตำแหน่งทางวิชาการ
     * สังกัดภาควิชา
   - ระยะเวลาดำเนินการ
     * วันที่เริ่มต้น
     * วันที่คาดว่าจะแล้วเสร็จ
     * แผนการดำเนินงาน (Gantt Chart)

2. **คพ.02 - แบบฟอร์มขอสอบโครงงานพิเศษ**
   - ข้อมูลการสอบ
     * ประเภทการสอบ (โครงงานพิเศษ 1/ปริญญานิพนธ์)
     * วันที่ขอสอบ
     * เวลาที่ขอสอบ
     * สถานที่สอบ
   - ความคืบหน้าโครงงาน
     * เปอร์เซ็นต์ความคืบหน้า
     * ส่วนที่ดำเนินการแล้ว
     * ส่วนที่กำลังดำเนินการ
     * ส่วนที่ยังไม่ได้ดำเนินการ
   - เอกสารแนบ
     * บทที่ 1-3 (สำหรับปริญญานิพนธ์)
     * Source Code
     * สมุดบันทึกการให้คำปรึกษา

3. **คพ.04 - หนังสือรับรองการทดสอบโครงงาน**
   - ข้อมูลการทดสอบ
     * วันที่ทดสอบ
     * สถานที่ทดสอบ
     * ผู้เข้าร่วมการทดสอบ
   - รายละเอียดการทดสอบ
     * ฟังก์ชันที่ทดสอบ
     * วิธีการทดสอบ
     * ผลลัพธ์ที่คาดหวัง
     * ผลลัพธ์ที่ได้
   - ผลการประเมิน
     * ความถูกต้องของระบบ
     * ประสิทธิภาพการทำงาน
     * ความเสถียร
     * ความปลอดภัย
   - ข้อเสนอแนะจากผู้ทดสอบ

4. **สมุดบันทึกการให้คำปรึกษาโครงงานพิเศษ**
   - ข้อมูลการให้คำปรึกษา
     * ครั้งที่
     * วันที่
     * เวลาเริ่ม-สิ้นสุด
     * รูปแบบ (พบที่ห้องพัก/ออนไลน์)
   - หัวข้อที่ปรึกษา
     * ประเด็นที่นำเสนอ
     * ปัญหาที่พบ
     * การแก้ไขที่ผ่านมา
   - ข้อเสนอแนะจากอาจารย์
     * คำแนะนำด้านเทคนิค
     * แนวทางการแก้ปัญหา
     * เอกสารอ้างอิง
   - การติดตามผล
     * งานที่ต้องทำต่อ
     * กำหนดส่ง
     * นัดหมายครั้งถัดไป
