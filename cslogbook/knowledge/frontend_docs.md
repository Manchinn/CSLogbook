# ขั้นตอนการใช้งานระบบฝึกงาน CSLogbook (สำหรับนักศึกษา)

ภาพรวมการใช้งานระบบฝึกงานของ CSLogbook ในส่วนที่นักศึกษาจะได้ใช้งานโดยตรง

## ภาพรวมขั้นตอนการใช้งานระบบฝึกงานสำหรับนักศึกษา:

1.  **ขั้นตอนที่ 1: ตรวจสอบคุณสมบัติและข้อกำหนดเบื้องต้น**
    *   **เริ่มต้นจาก:** นักศึกษาจะต้องเข้ามาตรวจสอบที่หน้าประวัตินักศึกษาก่อน ว่าตนเองมีคุณสมบัติครบถ้วนตามที่หลักสูตรกำหนดไว้สำหรับการฝึกงานหรือไม่ เช่น จำนวนหน่วยกิตสะสม จำนวนหน่วยกิตภาควิชาที่เรียนผ่านแล้ว
    *   **ระบบจะแสดงอะไร?:** ในหน้าประวัติการศึกษาจะแสดง
        *   รายละเอียดข้อมูลหน่วยกิต: ระบบอาจมีการตรวจสอบสิทธิ์เบื้องต้นของนักศึกษาด้วย
        *   ไทม์ไลน์การศึกษา : ขั้นตอนของระบบฝึกงาน และ โครงงานพิเศษ

2.  **ขั้นตอนที่ 2: ยื่นคำร้องขอฝึกงาน (กรอกแบบฟอร์ม คพ.05)**
    *   **เริ่มต้นจาก:** เมื่อตรวจสอบคุณสมบัติผ่านแล้ว นักศึกษาก็จะเข้ามาที่ส่วนของการกรอกแบบฟอร์มคำร้องขอฝึกงาน (คพ.05)
    *   **ต้องกรอกอะไรบ้าง?:** จะเป็นการกรอกข้อมูลส่วนตัว ข้อมูลของสถานประกอบการที่จะไปฝึกงาน ผู้ประสานงาน(พี่เลี้ยงฝึกงาน/หัวหน้างาน) และระยะเวลาการฝึกงาน ต้องแนบไฟล์เอกสารเพิ่มเติม เช่น ใบแสดงผลการเรียน (Transcript) ด้วย
    *   **ข้อมูลจะไปไหน?:** เมื่อกรอกครบถ้วนแล้ว ข้อมูลจะถูกส่งไปให้เจ้าหน้าภาควิชาพิจารณา

3.  **ขั้นตอนที่ 3: รอการอนุมัติและเตรียมตัวเริ่มฝึกงาน**
    *   **เริ่มต้นจาก:** หลังจากยื่นคำร้อง คพ.05 ไปแล้ว นักศึกษาสามารถเข้ามาติดตามสถานะการอนุมัติได้ในระบบ
    *   **ระบบจะแสดงอะไร?:** ในหน้ากรอกแบบฟอร์มคำร้องขอฝึกงาน (คพ.05) จะแสดงสถานะเป็น อนุมัติ และ สามารถตรวจสอบข้อมูลสถานประกอบการได้

4.  **ขั้นตอนที่ 4: บันทึกการปฏิบัติงานประจำวัน (Logbook)**
    *   **เริ่มต้นจาก:** เมื่อคำร้อง คพ.05 ได้รับการอนุมัติแล้ว และนักศึกษาได้เริ่มฝึกงาน ก็จะต้องเข้ามาบันทึกรายละเอียดการปฏิบัติงานในแต่ละวัน 
    *   **ต้องบันทึกอะไรบ้าง?:** โดยทั่วไปจะบันทึกว่าในแต่ละวันได้ทำงานอะไรบ้าง ใช้เวลากี่ชั่วโมง พบปัญหาอะไร และแก้ไขอย่างไร
    *   **ข้อควรรู้:** หากนักศึกษายังไม่ได้ยื่น คพ.05 หรือยังไม่ได้รับอนุมัติ เมื่อพยายามเข้าหน้านี้ ระบบอาจจะนำทางนักศึกษากลับไปที่หน้ายื่น คพ.05 ก่อน

5.  **ขั้นตอนที่ 5: สรุปผลการฝึกงานและประสบการณ์ (Reflection)**
    *   **คุณลูกค้าจะทำอะไร?:** เมื่อฝึกงานเสร็จสิ้นแล้ว นักศึกษาจะต้องเข้ามาเขียนบทสรุปและประสบการณ์ที่ได้รับจากการฝึกงาน
    *   **ต้องเขียนอะไรบ้าง?:** จะเป็นหัวข้อเกี่ยวกับสิ่งที่ได้เรียนรู้ ประสบการณ์และทักษะสำคัญที่ได้รับ การนำความรู้ไปประยุกต์ใช้ในอนาคต และสิ่งที่คิดว่าควรปรับปรุงหรือพัฒนาตนเอง หากเคยบันทึกไว้แล้วก็สามารถเข้ามาแก้ไขได้

6.  **ขั้นตอนที่ 6: ตรวจสอบภาพรวมและดาวน์โหลดเอกสาร**
    *   **คุณลูกค้าจะทำอะไร?:** นักศึกษาสามารถเข้ามาดูภาพรวมทั้งหมดของการฝึกงานได้ที่หน้าสรุปผล
    *   **ระบบจะแสดงอะไร?:** หน้านี้จะแสดงข้อมูลสถิติต่างๆ เช่น จำนวนชั่วโมงการทำงานทั้งหมด ความคืบหน้า และมีแท็บให้เลือกดูข้อมูลในมุมมองต่างๆ เช่น รายการบันทึก ทักษะที่ได้รับ และบทสรุปที่นักศึกษาเขียนไว้
    *   **ดาวน์โหลดเอกสาร:** ที่สำคัญคือ นักศึกษาสามารถดาวน์โหลดเอกสารสรุปผลการฝึกงานเป็นไฟล์ PDF ได้จากหน้านี้เลย 

7.  **ขั้นตอนที่ 7: การประเมินผลจากพี่เลี้ยง (ในมุมมองนักศึกษา)**
    *   **คุณลูกค้าอาจเกี่ยวข้องอย่างไร?:** ในหน้าสรุปผลการฝึกงาน อาจมีส่วนที่ให้นักศึกษาส่งแบบฟอร์มประเมินผลการฝึกงานไปยังพี่เลี้ยงที่สถานประกอบการ นักศึกษาอาจจะเห็นสถานะว่าแบบฟอร์มถูกส่งไปแล้วหรือยัง
    *   **ส่วนของพี่เลี้ยง:** การกรอกแบบฟอร์มประเมินจริงๆ จะเป็นหน้าที่ของพี่เลี้ยง ซึ่งจะมีหน้าสำหรับกรอกข้อมูลโดยเฉพาะ และนักศึกษาไม่ได้เข้าไปใช้งานส่วนนี้โดยตรง แต่เป็นส่วนหนึ่งของกระบวนการ

## ส่วนประกอบอื่นๆ ที่เกี่ยวข้องในระบบ:

*   **เมนูนำทาง (Sidebar):** ช่วยให้นักศึกษาเข้าถึงส่วนต่างๆ ของระบบฝึกงานได้ง่ายขึ้น
*   **การแสดงไฟล์ PDF:** หากมีการแสดงตัวอย่างเอกสารในระบบ ก็จะมีส่วนที่ใช้แสดงไฟล์ PDF 
*   **การแสดงข้อมูลนักศึกษา:** ข้อมูลพื้นฐานของนักศึกษาที่แสดงในส่วนต่างๆ ก็จะดึงมาจากโปรไฟล์ของนักศึกษาเอง
