# รายงานสถานะการใช้งานระบบ SSO — CSLogbook

**วันที่จัดทำ:** 7 เมษายน 2569
**จัดทำโดย:** ทีมพัฒนาระบบ CSLogbook
**เสนอต่อ:** คณะกรรมการพิจารณาระบบสารสนเทศ

---

## 1. สถานะปัจจุบันของระบบ

ระบบ CSLogbook ใช้การลงชื่อเข้าใช้งานผ่าน **SSO ของมหาวิทยาลัยเทคโนโลยีพระจอมเกล้าพระนครเหนือ (ICIT Account)** เป็นช่องทางหลักสำหรับการยืนยันตัวตน โดย ณ ปัจจุบัน ระบบ SSO อยู่ในสถานะ **Development Mode (โหมดพัฒนา)** ซึ่งได้รับอนุญาตจาก ICIT ให้ใช้งานเพื่อการพัฒนาและทดสอบระบบเท่านั้น

## 2. ขอบเขตการใช้งานใน Development Mode

ภายใต้ Development Mode ระบบ SSO มีข้อจำกัดดังนี้

**ผู้ที่สามารถเข้าใช้งานผ่าน SSO ได้:**
- นักศึกษาที่มี ICIT Account เท่านั้น

**ผู้ที่ยังไม่สามารถเข้าใช้งานผ่าน SSO ได้:**
- อาจารย์ที่ปรึกษา — ใช้ระบบ login ด้วย username/password ภายในระบบแทน
- เจ้าหน้าที่ภาควิชา — ใช้ระบบ login ด้วย username/password ภายในระบบแทน
- ผู้ควบคุมงาน (Supervisor) ภายนอก — เข้าใช้ผ่านลิงก์เฉพาะที่ระบบจัดส่งให้

ข้อจำกัดนี้เป็นเงื่อนไขจาก ICIT ที่กำหนดให้ Development Mode รองรับเฉพาะบัญชีนักศึกษา ไม่ใช่ข้อจำกัดจากตัวระบบ CSLogbook เอง

## 3. เหตุผลที่ระบบยังอยู่ใน Development Mode

การเปิดใช้งาน SSO ในระดับ **Production Mode** ต้องได้รับการอนุมัติจาก ICIT อย่างเป็นทางการ โดยมีขั้นตอนดังนี้

1. **การพิจารณาจากคณะกรรมการ** — ระบบต้องผ่านการพิจารณาและได้รับความเห็นชอบจากคณะกรรมการที่เกี่ยวข้องก่อน
2. **การยื่นคำร้อง** — เมื่อได้รับความเห็นชอบแล้ว จะดำเนินการยื่นคำร้องขออนุมัติใช้งาน Production Mode ต่อผู้ดูแลระบบ ICIT
3. **การตรวจสอบและอนุมัติ** — ICIT พิจารณาคำร้อง ตรวจสอบความพร้อมของระบบ และอนุมัติการใช้งาน

## 4. ผลกระทบเมื่อได้รับอนุมัติ Production Mode

เมื่อระบบ SSO ได้รับการอนุมัติเป็น Production Mode จะส่งผลให้

- **อาจารย์และบุคลากร** สามารถลงชื่อเข้าใช้งานผ่าน ICIT Account ได้โดยตรง ไม่ต้องจำ username/password แยกต่างหาก
- **ความปลอดภัย** ได้รับการยกระดับ เนื่องจากใช้ระบบยืนยันตัวตนของมหาวิทยาลัยเป็นหลัก
- **การจัดการบัญชีผู้ใช้** ง่ายขึ้น ลดภาระของเจ้าหน้าที่ในการสร้างและดูแลบัญชีภายในระบบ

## 5. กำหนดการ

ขณะนี้ยังไม่มีกำหนดการที่แน่ชัดสำหรับการยื่นขอ Production Mode เนื่องจากต้องรอผลการพิจารณาจากคณะกรรมการเป็นลำดับแรก ทีมพัฒนาพร้อมดำเนินการยื่นคำร้องทันทีเมื่อได้รับความเห็นชอบ

## 6. สรุป

| หัวข้อ | รายละเอียด |
|--------|-----------|
| สถานะปัจจุบัน | Development Mode |
| ผู้ใช้งาน SSO ได้ | นักศึกษาเท่านั้น |
| ผู้ใช้งานอื่น | Login ด้วย username/password ภายในระบบ |
| เงื่อนไขเปิด Production | ผ่านการพิจารณาจากคณะกรรมการ → ยื่นคำร้องต่อ ICIT |
| กำหนดการ | รอผลพิจารณาจากคณะกรรมการ |
